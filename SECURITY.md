## Response Process
- **Acknowledgement:** within 7 days  
- **Assessment:** validation + risk analysis  
- **Mitigation:** patch development  
- **Release:** security update + documentation update  
---
## Security Scope
Includes:
- Spreadsheet automation engine  
- WQI computation modules  
- Web platform  
- Data processing & visualization  
- Future GIS/QGIS modules  
- API and cloud services  
---
## Out of Scope
- Social engineering attacks  
- Third-party infrastructure issues  
- DoS requiring extreme resources  
- External dependencies not introduced by HYDROQUALIX-4™  
---
## Security Principles
- Data minimization  
- Scientific transparency  
- Reproducibility  
- Secure development lifecycle  
- Responsible data handling  
---
## Responsible Disclosure
- No unauthorized access or data modification  
- No service disruption  
- Allow time for remediation  
- Good-faith testing is permitted
